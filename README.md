# Controls

> [!WARNING]
> Controls are subject to change.

| Button | Action |
|-------------------------------|----------------------------------------------------------------|
| Move mouse | Look around |
| Up, Down, Left, Right | Move |
| F1 | Generate humans |
| 1, 2, 3, 4, 5, 6... | Move between maps |
